Problem 9: Special Pythagorean Triplet

A Pythagorean triplet is a set of three natural numbers, \( a < b < c \), for which,

[a^2 + b^2 = c^2]

For example,

[3^2 + 4^2 = 9 + 16 = 25 = 5^2]

There exists exactly one Pythagorean triplet for which ( a + b + c = 1000).

Find the product (abc).


 مسئله ۹: سه‌تایی فیثاغورثی ویژه

یک سه‌تایی فیثاغورثی مجموعه‌ای از سه عدد طبیعی مانند ( a < b < c ) است که در آن:

[a^2 + b^2 = c^2]

برای مثال:

[3^2 + 4^2 = 9 + 16 = 25 = 5^2]

دقیقاً یک سه‌تایی فیثاغورثی وجود دارد که در آن ( a + b + c = 1000 ).

حاصلضرب ( abc) را پیدا کنید.
